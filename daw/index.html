		welcome to my cheap DAW! Here's how to use it:
		1. Search for a sample (powered by freesound.org)
		2. Add to the sequencer or QWERTY keys using the dropdown next to each sample 
		3. Click 'Sequencer' or 'QWERTY' to select which view you want to use
		4. Use the play controls at the bottom to control your playback
		5. Enjoy!
